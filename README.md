# 📝 Notepad Java

A simple text editor application built using Java that recreates the core functionality of a traditional Notepad application. This project allows users to create, edit, save, and manage text files through a clean desktop interface.

The project was built to strengthen understanding of Java GUI development, event handling, file management, and desktop application architecture.

## ✨ Features

- Create new text files
- Open existing files
- Save files
- Save As functionality
- Edit text content
- Cut, Copy, and Paste support
- Simple and user-friendly interface
- Basic file handling operations
- Lightweight desktop application

## 🛠️ Technologies Used

- Java
- Java Swing / AWT
- File I/O
- Event Listeners

## 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Sappraise123/Notepad_java.git
```

Move into the project directory:

```bash
cd Notepad_java
```

Compile the Java files:

```bash
javac *.java
```

Run the application:

```bash
java Main
```

If using an IDE:

- Open the project in IntelliJ IDEA, Eclipse, or NetBeans
- Build the project
- Run the main class
